1. COMPONENT OVERVIEW
─────────────────────

Radio buttons let users select exactly one option from a mutually
exclusive set. Best suited for short lists of 2–5 options where
seeing every choice reduces decision-making effort.

Component type: Form input (single selection)
Interaction: Click / keyboard (arrow keys within group, tab exits)

────────────────────────────────────────

2. ANATOMY
──────────

  ┌──────────────────────────────────┐
  │  ( ● )  Label text               │
  └──────────────────────────────────┘

  1. Radio circle — circular border, filled inner dot when selected
  2. Label — text describing the option, clicking selects the radio

────────────────────────────────────────

3. PROPERTIES
─────────────

| Prop           | Type                    | Default   | Description                        |
|----------------|-------------------------|-----------|------------------------------------|
| size           | "md" | "sm"             | "md"      | Size variant                       |
| side           | "left" | "right"        | "left"    | Radio position relative to label   |
| label          | string                  | "Label"   | Label text                         |
| name           | string                  | undefined | Groups radios for mutual exclusion |
| checked        | boolean                 | undefined | Controlled checked state           |
| defaultChecked | boolean                 | undefined | Uncontrolled initial state         |
| disabled       | boolean                 | false     | Disables the radio                 |
| onChange       | (e) => void             | undefined | Change handler                     |
| className      | string                  | undefined | Additional CSS classes              |

Extends: InputHTMLAttributes<HTMLInputElement> (except "size" | "type")

8. LAYOUT
─────────

  Root: <label> flex items-center
  Hidden input: sr-only (visually hidden, accessible)
  Circle: shrink-0, flex center, rounded-full, border-2
  Inner dot: rounded-full, conditionally rendered
  Label: flex-1, min-w-0
  Side swap: flex order changes based on side prop

────────────────────────────────────────

9. ACCESSIBILITY
────────────────

  - Uses native <input type="radio"> with name grouping
  - Hidden via sr-only but remains in accessibility tree
  - Arrow keys navigate between radios in same group
  - Tab key exits the radio group
  - Disabled uses native disabled attribute
  - Label wraps the input for click-to-select

────────────────────────────────────────

10. CONTROLLED / UNCONTROLLED SUPPORT
─────────────────────────────────────

  Controlled mode:
    - Pass `checked` prop — component reflects external state
    - Must provide `onChange` handler to update state
    - Recommended for radio groups

  Uncontrolled mode:
    - Pass `defaultChecked` prop for initial state
    - Internal state tracks changes via `useState`
    - Browser manages via name grouping
