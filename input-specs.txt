1. COMPONENT OVERVIEW
─────────────────────

Input lets users enter and edit short-form text. It is the most
common form control and the foundation of data collection. Each
input has a visible label, an editable field, and optional
helper/error text. Leading and trailing icons provide context.

Component type: Form input
Interaction: Text entry (keyboard)

────────────────────────────────────────

2. ANATOMY
──────────

  ┌────────────────────────────────┐
  │  Label                        │
  ├────────────────────────────────┤
  │ [icon] Placeholder       [icon]│
  ├────────────────────────────────┤
  │  Helper text / Error text     │
  └────────────────────────────────┘

  1. Label — descriptive text above the field (linked via htmlFor)
  2. Input container — bordered box with padding
  3. Input text / placeholder — the editable area
  4. Leading icon (optional) — left icon inside input
  5. Trailing icon (optional) — right icon inside input
  6. Helper / Error text — guidance or error below the field

────────────────────────────────────────

3. PROPERTIES
─────────────

| Prop         | Type                         | Default    | Description                        |
|--------------|------------------------------|------------|------------------------------------|
| size         | "lg" | "md" | "sm"           | "md"       | Size variant                       |
| state        | "default" | "error"          | "default"  | Validation state                   |
| label        | string                       | undefined  | Label text above the input         |
| helperText   | string                       | undefined  | Helper text below the input        |
| errorMessage | string                       | undefined  | Error text (replaces helper)       |
| leadingIcon  | ReactNode                    | undefined  | Left icon inside the input         |
| trailingIcon | ReactNode                    | undefined  | Right icon inside the input        |
| placeholder  | string                       | undefined  | Placeholder text                   |
| disabled     | boolean                      | false      | Disables the input                 |
| id           | string                       | auto-gen   | HTML id for label association      |
| className    | string                       | undefined  | Additional CSS classes             |

Extends: InputHTMLAttributes<HTMLInputElement> (except "size")

7. LAYOUT
─────────

────────────────────────────────────────

8. ACCESSIBILITY
────────────────

  - Label linked via htmlFor → id (auto-generated if not provided)
  - Helper/error linked via aria-describedby
  - Error state: aria-invalid="true" on <input>
  - Error message: role="alert" for screen reader announcements
  - Focus ring: 3px shadow (shadow-focus)
  - Disabled uses native disabled attribute
