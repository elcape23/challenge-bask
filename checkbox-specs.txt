1. COMPONENT OVERVIEW
─────────────────────

Checkbox Group pairs a square checkbox indicator with a text label.
It supports three visual states (unchecked, checked, indeterminate),
two sizes (md, sm), and two label positions (left, right).

Component type: Form input
Interaction: Toggle (click / space key)

────────────────────────────────────────

2. ANATOMY
──────────

  side=left (default)
  ┌──────────────────────────────────┐
  │  ┌──┐                           │
  │  │✓ │  Label text               │
  │  └──┘                           │
  └──────────────────────────────────┘

  side=right
  ┌──────────────────────────────────┐
  │                           ┌──┐  │
  │  Label text               │✓ │  │
  │                           └──┘  │
  └──────────────────────────────────┘

  1. Checkbox Group — root flex container (row, items-center)
  2. .checkbox — square indicator sub-component
  3. Label — descriptive text

7. LAYOUT
─────────

  Direction:    row (horizontal)
  Alignment:    items-center
  Side=left:    checkbox → label
  Side=right:   label → checkbox
  Checkbox:     shrink-0 (does not shrink)
  Label:        flex-1 (fills remaining space)

────────────────────────────────────────

8. ACCESSIBILITY
────────────────

  - Uses native <input type="checkbox"> (visually hidden, sr-only)
  - Wrapped in <label> so clicking anywhere toggles the state
  - Sets aria-checked="mixed" for indeterminate state
  - Groups should use <fieldset> + <legend>
  - Keyboard: Space to toggle, Tab to move between checkboxes
  - Focus ring: 3px shadow (--shadow-focus)
